# 操作方式
## build
- clone the repository
- cd Mid-Project
- npm install
- npm start

under the other shell
- node server.js
## run

- 到首頁增加PDF網址，若不是PDF則只有部分網頁同意內嵌
- 到CLass看新增的選單，點進去之後可以觀賞自己貼的slide，右側可以問問題
- Class清單可以管理所有的Slide
